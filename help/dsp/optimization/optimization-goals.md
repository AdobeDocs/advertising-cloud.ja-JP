# 最適化目標とその使用方法

| 最適化目標 | 説明 | この目標を使用するタイミング |
| -----------| ---------- | ---------- |
| [!UICONTROL Always Max Bid & Highest Clickthrough Rate] | パッケージレベルの最適化では、予算配分は、クリックスルー率が最も高い配置に優先順位を付けます。<br><br>オークション評価は、支出目標が満たされている場合にCTRを優先します。送信された入札は常に[!UICONTROL Max Bid]に設定されますが、配置が成功すると、予測されるCTRしきい値はより厳しくなります。 | キャンペーンタイプ：ブランディング<br><br>ベンチマーク：最も高いクリックスルー率<br><br>広告タイプ：プリロール、ディスプレイ<br><br><b>注意：</b>超える必要のない固定CPM目標と、最大化する必要があるCTR目標がある場合に、この目標を使用します。 最高入札額を目的のCPM目標に設定すると、DSPは最高のCTRを達成しながら、予算を最大限に使用します。 |
| [!UICONTROL Always Max Bid & Highest Completion Rate] | パッケージレベルの最適化では、予算配分は、最も高い完了率を持つ配置に優先順位を付けます。<br><br>オークション評価では、支出目標が満たされている場合に完了率が優先されます。送信される入札は常に「最大入札額」に設定されますが、配置が優れている場合は、予測される完了率しきい値がより厳しくなります。 | キャンペーンタイプ：ブランディング<br><br>ベンチマーク：最も高い完了率<br><br>広告タイプ：プリロールのみ<br><br><b>注意：</b>超える必要のない固定CPM目標と、最大化する必要がある完了率目標がある場合に、この目標を使用します。 目的のCPM目標に最高入札額を設定すると、DSPは予算全体を費やしながら、可能な限り最高の完了率を達成します。 |
| [!UICONTROL Always Max Bid & Highest Engagement Rate] | パッケージレベルの最適化では、予算配分は、最も高いエンゲージメント率を持つ配置に優先順位を付けます。<br><br>オークション評価では、支出目標が満たされている場合に、エンゲージメント率が優先されます。送信される入札は常に「最高入札額」に設定されますが、プレースメントが優れている場合は、予測されるエンゲージメント率のしきい値がより厳しくなります。 | キャンペーンタイプ：ブランディング<br><br>ベンチマーク：最も高いエンゲージメント率<br><br>広告タイプ：モバイルインタースティシャルのみ<br><br><b>注意：</b>超える必要のない固定CPM目標と、最大化する必要があるエンゲージメント目標がある場合に、この目標を使用します。 目的のCPM目標に最高入札額を設定すると、DSPは予算を最大限に使用しながら、可能な限り最高のエンゲージメント率を達成します。 |
| [!UICONTROL Always Max Bid & Highest Viewability Rate (Adobe – GroupM)] | パッケージレベルの最適化では、予算配分は、視認性が最も高い配置に優先順位を付けます。<br><br>オークション評価では、支出目標が満たされている場合に視認性レートが優先されます。送信される入札は常に「最大入札額」に設定されますが、配置が優れていると、予測視認性率のしきい値がより厳しくなります。 | キャンペーンタイプ：ブランディング<br><br>ベンチマーク：最も高い視認性率<br><br>広告タイプ：インタラクティブプリロールのみ<br><br><b>注意：</b>この目標は、常にプレースメントレベルの最大入札額を使用します。<br><br>キャンペーンの「視認性の感度」設定が「標準（連続して2秒間表示される広告の50%）」に設定されている場合は、メディア評価評議会(MRC)の視認性の測定標準がキャンペーンに使用されます。キャンペーンが「厳密(視聴とオーディオの100%（50%持続時間）」に設定されている場合は、GroupMの視認性測定標準がキャンペーンに使用されます。 理想的には、キャンペーン設定を最適化目標および事前入札フィルター設定と一致させる必要があります。 |
