# SSPパートナー

サプライサイドプラットフォーム(SSP)は、デジタルメディアの所有者やパブリッシャーが自動オークションでデジタル広告を販売するのに役立つ広告テクノロジープラットフォームです。 デジタルメディアの所有者と発行者は、SSPを使用して、広告インベントリの提供と配布を調整、管理、収益化します。 Advertising Cloud DSPは、すべての主要なSSPと統合されており、広告主が広告ニーズに合わせて高品質のインベントリに簡単にアクセスできます。
