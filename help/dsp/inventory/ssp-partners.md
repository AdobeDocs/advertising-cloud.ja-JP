# SSP パートナー

サプライサイドプラットフォーム (SSP) は、デジタルメディアの所有者やパブリッシャーが自動オークションでデジタル広告を販売するのに役立つ広告テクノロジープラットフォームです。 デジタルメディアの所有者とパブリッシャーは、SSP を使用して、広告インベントリの提供と配布を調整、管理、収益化します。 Advertising Cloud DSPは、すべての主要な SSP と統合されており、広告主が広告ニーズに応じて高品質のインベントリに簡単にアクセスできます。
